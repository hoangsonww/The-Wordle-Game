# Screenshots Directory

This directory contains screenshots of the Game Hub application.

## Required Screenshots

To complete the documentation, please add the following screenshots:

1. **home.png** - Home page showing all 4 games
2. **wordle.png** - Wordle game in progress
3. **connections.png** - Connections game showing the word grid
4. **sudoku.png** - Sudoku game with the 9x9 grid
5. **numbers.png** - Numbers game with operations

## How to Take Screenshots

1. Run the development server: `npm run dev`
2. Navigate to each game
3. Take screenshots at appropriate game states
4. Save as PNG files with the names above
5. Update this README once screenshots are added

## Recommended Tools

- **macOS**: Cmd + Shift + 4
- **Windows**: Windows + Shift + S
- **Linux**: Screenshot tool or `gnome-screenshot`

## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG for best quality
- **Content**: Show the game in an interesting state (mid-game, not empty)
- **Consistency**: Use the same browser and viewport size for all screenshots
